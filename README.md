# Jam with NVDA v1.0-beta

A professional music player add-on for NVDA, powered by the BASS audio engine.

## Features

- **Smart Background Play**: Continue listening even after closing the player interface.
- **Accessible Seeking**: Navigate through your tracks with precise keyboard shortcuts.
- **Smart Volume**: Fine-tuned volume control with persistence across sessions.
- **Architecture Aware**: Seamlessly supports both 32-bit and 64-bit NVDA versions.

## Shortcuts

- **NVDA+Shift+J**: Open Player / Focus Player
- **NVDA+J**: Global Play/Pause (Background)
- **Space**: Play/Pause (Inside dialog)
- **Left/Right Arrows**: Seek -5 / +5 seconds
- **Shift + Left/Right Arrows**: Seek -30 / +30 seconds
- **Up/Down Arrows**: Volume Up/Down (5%)
- **Shift + Up/Down Arrows**: Volume Up/Down (10%)
- **T**: Announce current playback time
- **Alt+L**: Load Song
- **Alt+S**: Stop
- **Alt+U**: Mute/Unmute
- **Alt+M / Escape**: Minimize (Keep playing in background)
- **Alt+X**: Stop and Exit Jam
