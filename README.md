\# Jam with NVDA 🎵



\*\*Jam with NVDA\*\* is a standalone music player add-on designed specifically for NVDA screen-reader users. This project was born out of a personal passion for music and a desire to create a simpler, more accessible way to enjoy audio directly within NVDA.



\---



\### ✨ Key Features



\* \*\*BASS Engine Integration:\*\* Uses the BASS audio library for reliable sound quality and performance.

\* \*\*Standalone Playback:\*\* No need for heavy external players like VLC or Windows Media Player.

\* \*\*Minimalist Design:\*\* A clean, clutter-free interface built for efficiency with screen readers.

\* \*\*Background Mode:\*\* Your music keeps playing smoothly while you work.

\* \*\*Lightweight:\*\* Designed to be fast and have minimal impact on NVDA's resources.



\---



\### ⌨️ Keyboard Shortcuts



| Action | Shortcut |

| :--- | :--- |

| \*\*Open Player\*\* | `NVDA + Shift + J` |

| \*\*Play / Pause\*\* | `Space` |

| \*\*Volume Up\*\* | `Up Arrow` |

| \*\*Volume Down\*\* | `Down Arrow` |

| \*\*Seek Forward\*\* | `Right Arrow` |

| \*\*Seek Backward\*\* | `Left Arrow` |



\---



\### 📥 Installation



1\.  Go to the \[Releases](https://github.com/dev-vignesh3/jam-with-nvda/releases) page.

2\.  Download the latest `.nvda-addon` file.

3\.  Open the file and follow the NVDA prompts to install.

4\.  Restart NVDA to activate the add-on.



\---



\### 🛠️ Technical Stack



\* \*\*Language:\*\* Python 3

\* \*\*GUI Framework:\*\* wxPython

\* \*\*Audio Engine:\*\* BASS Library (via `ctypes`)

\* \*\*Compatibility:\*\* NVDA 2024.1 and above
